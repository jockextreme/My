Step-by-Step Tutorial

Type 1: Basic SSIDs

SSID Formats: PLDTHOMEDSL, PLDTmyDSLPAL, PLDmyDSLBiz

Example MAC: 00:1F:FB:0F:C5:A8

Password Format: PLDTWIFI + last 5 MAC characters

Result: PLDTWIFIFC5A8

Type 2: Numeric Ending SSIDs

SSID Format: PLDTHOMEDSLxxxxx (5 numbers)

Example: PLDTHOMEDSL21354

Calculation: 21354 × 3 = 64062

Password: pLDTWIFI64062

Type 3: Fiber with Underscore

SSID Format: PLDTHOMEFIBR_xxxxxx

Example: PLDTHOMEFIBR_123abc

Hex Conversion: 1→e, 2→d, 3→c, a→5, b→4, c→3

Password: wlanedc543

Type 5: HomeBro Ultera

SSID Format: HomeBro_Ultera

Example MAC: 00:1F:FB:0F:C5:A8

Password Format: HomeBro_ + last 6 MAC characters

Result: HomeBro_0FC5A8

Important Notes

  • MAC address can often be found on the router sticker
  • Changed passwords won't be shown by this tool
  • Always respect network privacy laws
